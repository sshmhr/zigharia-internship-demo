Instructions

Below are the steps to run the demo of SensifyID Javascript SDK demo application.

  1. Click Start Tracking button.
  2. Go to the blue bordered area and interact with the form. (It is a sample form for demo and it will not save/post any content that you type.)
  3. Once done, click the Stop Tracking button in the top.
  4. Now your tracking session is completed.
  5. Click on the View Results button to view the data which is collected using the SDK.
